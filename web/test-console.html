Console Test Instructions

  1. Go to http://localhost:3001/wtaf-landing
  2. Scroll to the very bottom of the page
  3. Wait 1 second for the "▚ dev" handle to appear in bottom left
  4. Click on "▚ dev" to open the console
  5. Try these commands:
    • help - Shows all commands including new auth commands
    • login - Opens login form in console
    • signup - Opens signup form in console
    • whoami - Shows current user
    • wtaf make a todo app - Requires authentication
