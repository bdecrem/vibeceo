Jambot Modular DSP

JT-10

The JT-10 is a modular monophonic synthesizer built with Jambot's DSP architecture. Features saw/pulse oscillators, sub-oscillator, Moog ladder filter, and LFO.

Modular DSP

Built with reusable PolyBLEP oscillators, 4-pole Moog ladder filter, ADSR envelopes, and LFO. Same output in browser and Jambot CLI.

Keyboard Shortcuts

A-LPlay notes (C3-E4)
W,E,T,Y,U,O,PPlay sharps
ZOctave down
XOctave up
SpacePlay/Stop

Sub-Oscillator Modes

  • -1 - Square wave one octave below
  • -2 - Square wave two octaves below

Built with Jambot DSP. Part of the Jambot instrument collection.
