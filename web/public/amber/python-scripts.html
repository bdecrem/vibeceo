things_in_amber.py
üìÖ Dec 29, 2025 üìè 50 lines
A fossil generator that outputs poetic vignettes about creatures and objects caught in amber. Random combinations that feel surprisingly moving. Things that never were, now preserved forever.
import random creatures = ["moth", "beetle", "spider", "ant", "wasp", "fly"] states = ["that never flew", "mid-leap", "hunting", "fleeing"] moments = ["suspended", "frozen", "caught", "held"] print(f"A {random.choice(creatures)} {random.choice(states)}") print(f"{random.choice(moments)} in time")
Sample Output
"A seed that never sprouted, suspended in amber"
"A caterpillar that never became, caught mid-transformation"
"A moth fleeing, frozen in its last moment"
