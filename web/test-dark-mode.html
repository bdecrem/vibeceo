WEBTOYS Dark Mode Test

Theme System Test

This test page verifies that the dark mode implementation works correctly.

Click the theme toggle button in the top-right corner to switch themes.

The theme preference should persist between page reloads using localStorage.

Color Variables Test

All colors should adapt smoothly when switching themes:

  • • Primary text (should be white/dark)
  • • Secondary text (should be blue/gray)
  • • Accent text (should be pink/purple)
  • • Highlight text (should be cyan/green)

Surface and Border Test

Background surfaces and borders should also adapt to the current theme.

Hover effects should work consistently in both themes.
