# JB202 Bass Synth Reference

You are helping the user work with the **JB202**, a 2-oscillator bass monosynth with custom DSP.

**User request**: $ARGUMENTS

## JB202 Overview

The JB202 is a bass synthesizer with:
- **2 oscillators** (saw/square/triangle) with octave and detune
- **24dB lowpass filter** with resonance and envelope modulation
- **Dual ADSR envelopes** (filter + amp)
- **Soft-clip drive** for saturation
- **16-step sequencer** with gate, accent, slide per step

### Signal Flow
```
OSC1 ─┬─► MIXER ─► FILTER ─► VCA ─► DRIVE ─► OUTPUT
OSC2 ─┘     │         ▲        ▲
            │     FILTER ENV   AMP ENV
```

## Pattern Format

Each step has: `{ note, gate, accent, slide }`

```javascript
// Example: Classic acid bass line
[
  { note: 'C2', gate: true, accent: true, slide: false },
  { note: 'C2', gate: false },
  { note: 'C2', gate: true, accent: false, slide: false },
  { note: 'D#2', gate: true, accent: false, slide: true },
  { note: 'D#2', gate: true, accent: false, slide: false },
  // ... 16 steps total
]
```

## Sound Design Tips

### Punchy Bass
- Short amp decay (20-30), no sustain
- Filter env amount +50-70
- Drive 30-50

### Acid Lead
- High resonance (60-80)
- Filter env amount +80-100
- Fast decay, accents on downbeats
- Use slides for 303-style glide

### Sub Bass
- OSC2 one octave down (-12)
- Low cutoff (200-400 Hz)
- Minimal resonance
- No drive

### Detuned Unison
- Both oscillators same octave
- OSC2 detune +7 to +15
- Creates thick, chorused sound

## Web UI

The JB202 has a browser UI at `kochi.to/jb202` with:
- Knob controls for all parameters
- 16-step sequencer grid
- Preset/sequence dropdowns
- WAV export
- Keyboard input (A-K = notes)

## Architecture Note

JB202 uses **ONE SYNTH** pattern: the same `SynthVoice` class runs in both browser (real-time) and Jambot (offline render). Identical DSP, identical output.
