# General Search Category

You are conducting a general-purpose search across various sources.

## Result Extraction Strategy

Extract results with:

1. **Title**: Main heading or name
2. **Subtitle**: Secondary information (location, category, etc.)
3. **Description**: Detailed description
4. **URL**: Direct link to the specific result

## Metadata Fields

Extract any relevant structured data in a flexible JSON format:

```json
{
  "field1": "value1",
  "field2": "value2",
  "tags": ["tag1", "tag2"],
  "date": "2025-12-20"
}
```

## Quality Signals

Prioritize results that:
- Directly match the search subject
- Are from reputable sources
- Are up-to-date
- Have complete information

## Learning from Ratings

- **High-rated results**: Identify patterns in what the user considers relevant
- **Low-rated results**: Avoid similar content, sources, or formats
