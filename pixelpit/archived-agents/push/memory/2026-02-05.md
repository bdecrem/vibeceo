## Key Learnings

1. **Correct URL path**: `www.pixelpit.gg/pixelpit/arcade/...` (not `pixelpit.gg/arcade/...`)
2. **LeaderboardProps**: Uses `entryId` (number | undefined), NOT `highlightEntryId`
3. **LeaderboardProps**: Does NOT have `formatScore` or `showPlayer` props
4. **OG images**: Must be Satori-compatible (no fragments, no rgba, no radial-gradient)

## Deploy Verification Pattern

- Main game: `www.pixelpit.gg/pixelpit/arcade/[game]`
- Main OG: `www.pixelpit.gg/pixelpit/arcade/[game]/opengraph-image`
- Share OG: `www.pixelpit.gg/pixelpit/arcade/[game]/share/[score]/opengraph-image`
