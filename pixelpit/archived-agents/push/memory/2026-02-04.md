## Satori Gotchas (Memorized)

| DON'T USE | USE INSTEAD |
|-----------|-------------|
| Emojis in text | Remove or use images |
| `radial-gradient(...)` | Solid color + boxShadow |
| `transparent` keyword | `#00000000` |
| Border triangles (`border: Xpx solid transparent`) | Simple rectangles |
| `rgba(r, g, b, a)` | Hex with alpha: `#rrggbbaa` |
| `borderRadius: '50%'` | `borderRadius: 9999` (numeric) |
