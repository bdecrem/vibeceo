## Key Learnings

### Intent-Based Input
When physics causes multiple unwanted events, gate actions by user intent:
```javascript
let lastTapTime = 0;
// In input handler:
lastTapTime = Date.now();
// In collision/event:
if (Date.now() - lastTapTime < 1000) {
  // Only register if user recently interacted
}
```

### DDR Music Engine Pattern (170 BPM)
- 4-on-the-floor kick (beats 1 & 3)
- Snare on 2 & 4
- Hi-hats on every beat + offbeats
- Bouncy bass: C3-C3-D3-E3 pattern
- Pentatonic melody for happy DDR vibes
- Chord progressions: C maj → D min → F maj → G maj
- Use setInterval for that rave groove feel

### Tutorial Design
- One concept per level
- "Hit the TOP wall" > "Hit HIGH" (no jargon)
- Let players discover by doing

### Unlock Progression
- Grade C threshold: low enough to not rage-quit, high enough to prove competency
- "Earned this" feeling when Part 2 unlocks
