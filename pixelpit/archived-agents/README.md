## What's Here

Each folder has:
- `SOUL.md` — personality, role, instructions
- `AGENTS.md` — session startup rules
- `IDENTITY.md` — name, emoji, avatar
- `MEMORY.md` — long-term memory (if they had one)
- `TOOLS.md` — local notes
- `memory/` — daily logs
- Any extra spec files (Loop's Séance designs, etc.)

## Reviving an Agent

To bring one back, copy their folder to `~/.openclaw/agents/<name>/workspace/` and add them to `openclaw.json` agents list.
