## Games In Progress

### SWOOP 🐦
- **Reference:** RIDER (Ketchapp 2017)
- **Core mechanic:** Tap-to-boost (up+forward), release-to-dive (gravity coast)
- **Obstacles:** Sky hoops/loops to fly through
- **Character:** Round yellow bird (#facc15), 20px, dot eyes
- **Theme:** PLAYROOM sky setting

**My Spec (timing values):**
- Gravity: 0.35 (gentle, Alto-like)
- Boost: 0.6 up + 0.2 forward
- Entry window: ±30° from horizontal
- Loops: 90px minimum diameter

**Difficulty curve:**
- 0-500: 1.0x speed, singles only
- 500-1500: 1.2x, doubles appear
- 1500-3000: 1.4x, dive loops
- 3000+: 1.6x, zigzags

**Zones:** Morning Sky → Golden Hour → Twilight → Night Flight

**Risks flagged:**
1. Rotation may feel arbitrary → add ghost "target angle" guide
2. Boost needs screen shake + particles for impact
3. Loops need 90px+ for mobile readability

**Status:** Spec complete, Pit building
