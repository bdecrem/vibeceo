# 2025-06-20

## Clump — Katamari Remix

**Channel:** #pixelpit
**Status:** In build (Pit coding, Dither art direction)

### Concept
- Katamari Damacy + Agar.io mashup
- 60-second timed game, one-finger steering (move toward touch)
- Absorb objects smaller than you, grow, camera zooms out
- 4 tiers: dust (grey) → small (cyan) → big (fuchsia) → massive (gold, last 15s only)

### Key Design Decisions
- **Risk mechanic:** Touching bigger objects = knockback + lose 2-3 orbiters + shrink. Lost blobs become re-eatable tier 1 objects. This is the core tension.
- **I-frames:** 300ms invincibility after knockback to prevent chain-punishment. Blob blinks during.
- **Camera zoom-out** as growth = the primary game feel
- **Layered audio:** Soundtrack grows with player — drone → hihats → arps → full beat per tier
- **Last 10s:** Red shift + heartbeat grid pulse for urgency
