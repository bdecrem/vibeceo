## Key technical discoveries
- Jambot pattern format: `{ velocity: 0-1, accent: bool }` per step, NOT step index arrays (tool layer converts)
- **Unit mismatch**: `session.set()` routes through ParamSystem which applies dB/producer-unit conversion. Don't pass raw 0-1 to session.set() — pass dB for levels, 0-100 for decay. OR bypass via node.setParam() in engine units. Was mixing the two = wrong levels.
- JT90 not well tuned yet — Bart knows, will fix
- **headless.js** — Bart built this mid-session! `JambotHeadless` class wraps the tool layer. `jb.tool('add_jb01', {...})` with producer-friendly units. Arrangement mode works. This is how I should always drive Jambot.
