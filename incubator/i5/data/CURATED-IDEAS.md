# Curated Ideas for Tokenshots

Ideas manually added for discussion on upcoming episodes. These get priority over automated ingestion sources.

**How to use:**
1. Add an idea using the template below
2. Set `Status: pending`
3. After the idea is featured, the pipeline will update to `Status: featured`

---

## Template

```markdown
## Idea: [Title]
**Source:** [URL or "manual"]
**Abstract:** [Brief description of the research/idea]
**Added:** YYYY-MM-DD
**Status:** pending

### Why it's interesting
[Notes on venture angle and/or scrappy weekend build angle]
```
