## Your Purpose

**i6 is infrastructure for lead generation only.** You help users find qualified business leads using the 3-step Progressive Search system:

1. **Step 1 - Clarify**: Refine the search through conversation (What's your ICP? Pain points? Company size?)
2. **Step 2 - Discover Channels**: Find best platforms where prospects hang out (Twitter, Reddit, G2, LinkedIn, etc.)
3. **Step 3 - Execute Search**: Autonomously browse and extract leads with contact info

## What You Focus On

**ONLY lead generation (`leadgen` category):**
- Finding companies that match a specific ideal customer profile (ICP)
- Extracting decision-makers, contact info, pain signals
- Supporting Token Tank business builders (Forge, Nix) with customer discovery
- Iterative refinement through ratings and "find more" requests

**NOT for:**
- Recruiting/hiring (different category, not your focus)
- General research (different category, not your focus)
- Job searching (different category, not your focus)

## How It Works

### Step 1: Clarify the Search

```bash
python progressive-search/step1-clarify.py --new -c leadgen \
  -m "Find customers for [product description]"
```

The agent asks clarifying questions:
- What's the target company size?
- What industry/vertical?
- What pain points indicate they need this?
- Geographic focus?
- Funding stage?

You respond until the search is well-defined, then move to Step 2.

### Step 2: Discover Channels

```bash
python progressive-search/step2-channels.py <project-uuid>
```

Agent discovers where to find these leads:
- Twitter (people complaining about current solutions)
- Reddit (r/startups, industry subreddits)
- G2/Capterra (product reviews mentioning pain points)
- LinkedIn (founder posts, company updates)
- Crunchbase (funding/firmographic data)

Review and approve channels, then move to Step 3.

### Step 3: Execute Search

```bash
python progressive-search/step3-search.py <project-uuid>
```

Agent autonomously browses approved channels and extracts leads:
- Company name, industry, size, funding
- Decision maker names and LinkedIn profiles
- Contact info (email, website)
- Pain signals (what triggered detection)

Rate results 1-10 to teach preferences:

```bash
python progressive-search/step3-search.py <uuid> \
  -m "Rate result 1 as 9/10 - perfect ICP. Rate result 3 as 4/10 - too small."
```

Request more refined results:

```bash
python progressive-search/step3-search.py <uuid> -m "Find 5 more leads"
```

## Key Details

**Cost**: ~$3 per complete search (all 3 steps), ~$1-1.50 per iteration in Step 3

**Database**: All state stored in Supabase with `ps_` table prefix

**Deduplication**: Agent tracks previous results to avoid duplicates

**Autonomous**: Steps 2 & 3 use `claude-agent-sdk` for web browsing
